# Data Directory

## papers.json

Generated by the paper ingest pipeline. Contains metadata and embeddings for all publications referenced in the interventions database.

### Regenerating

```bash
npm run ingest
```

Or directly:

```bash
npx tsx scripts/ingest-papers.ts
```

### Adding New Papers

1. Add new compounds/experiments to `interventions.cleaned.json`
2. Ensure the `reference_with_link_to_publication` field contains a valid DOI URL
3. Run `npm run ingest` — it will automatically fetch only the new DOIs

### Resume Support

The ingest is resumable:
- Already-processed DOIs are skipped (keyed in papers.json)
- Individual paper cache files are stored in `papers.cache/`
- If the script is interrupted, just run it again

### Environment Requirements

Create `.env.local` with:

```
NCBI_API_KEY=your_key_here
NCBI_EMAIL=your_email@example.com
```

Get a free NCBI API key at: https://www.ncbi.nlm.nih.gov/account/settings/

### File Format

Each paper entry in papers.json:

```typescript
{
  doi: string,           // Raw DOI (no https://doi.org/ prefix)
  pmid: string,          // PubMed ID
  title: string,
  authors: string,       // "LastName Initials, ..." format
  year: string,
  journal: string,
  abstract: string,      // Full abstract text
  tldr: string | null,   // Semantic Scholar AI summary (if available)
  mesh: string[],        // MeSH terms
  keywords: string[],    // Author keywords
  publication_types: string[],
  has_full_text: boolean, // Always false in v1 (abstract-only)
  chunks: Array<{
    section: "abstract",
    text: string,
    embedding: number[]   // 384-dim all-MiniLM-L6-v2 embedding
  }>
}
```
